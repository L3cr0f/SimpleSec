Para ejecutar el archivo "SimpleSec.jar" has de situarte con una terminal de comandos en el directorio en el que se encuentra y ejecutar el siguiente comando: "java -jar SimpleSec.jar".

Si desea visualizar los posibles argumentos que recibe, teclea "java -jar SimpleSec.jar h".
Cuando genera la clave para el cifrado del archivo "private.key" se genera un archivo HASH de dicha clave. Esto permite poder conocer cuando la clave es correcta y cuando no. Otro aspecto relacionado con dicha clave es el hecho de que usted no verá nada por pantalla, de modo que nadie pueda espiar lo que está escribiendo. 

El resto de opciones son como las mencionadas en el enunciado del proyecto.
